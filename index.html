Práctica 11: Terraform con Cloudflare

Infraestructura para el desarrollo continuo. Alejandro Rodríguez Veloquio

Objetivo

Esta práctica muestra cómo usar Terraform para crear infraestructura y conectarla con un proveedor de Cloudflare.

Elementos clave

  • Terraform como herramienta de Infraestructura como Código
  • Provider de Cloudflare para gestionar DNS y seguridad
  • Configuración declarativa y reproducible
